Deploy Hugo site on GitHub

Posted on March 3, 2017

Links: Hugo - Hosting on GitHub Personal/Organization Pages Deploy your blog to github pages automatically using Hugo and Travis We’ll create two separate repos, one for Hugo’s content, and a git submodule with the public folder’s content in it. Step by step: Create on GitHub <your-project>-hugo repository (it will host Hugo’s content) Create on GitHub <username>.github.io repository (it will host the public folder: the static website) git clone <<your-project>-hugo-url> && cd <your-project>-hugo Make your website work locally (hugo server --watch) Once you are happy with the results, Ctrl+C (kill server) and rm -rf public (don’t worry, it can always be regenerated with hugo) git submodule add -b master https://github.
